# ABAP Senior Showcase Revenue Report
Demo Projekt zur Präsentation von sauberem OO-ABAP, DI, Fehlerbehandlung, SALV & Tests.

## Quick Start
1) Repo via abapGit in SAP importieren  
2) Report ZREVENUE_CARRIER_REPORT ausführen  
3) ABAP Unit in Klasse ZCL_REV_SERVICE starten

## CI
- abaplint prüft Stil- & Syntax-Konventionen (Tooling-Konfig im Repo)
